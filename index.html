Portfolio

Commercial Killer

Commercial Killer is a user powered commercial detection app. It is essentially Ad Block Plus for one's television. It uses javascript, ruby on rails, and PSQL to function. The app has particular front end features like giphs, point counters, twitter modals, and badges. I created Commercial Killer in under a week. Hopefully one day you can see it in the app store!

Github Deployed site

Bucket Buddy

Bucket Buddy is an app created in a four person group at General Assembly. It has features like profiles, authentication, and social features like seeing all users' buckets. A bucket is a bucket list item that a user creates and the app keeps track of one's bucket list goals. Bucket Buddy uses express, javascript, and MongoDB for front and back end as well as server capabilities, respectively.
